# lifecycle

Describe life cycle of the widget.

## Getting Started
The lifecycle has the following simplified steps:
- createState()
- mounted == true
- initState()
- didChangeDependencies()
- build()
- didUpdateWidget()
- setState()
- deactivate()
- dispose()
- mounted == false
