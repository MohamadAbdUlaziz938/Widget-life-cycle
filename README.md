# lifecycle

Describe life cycle of the widget.

## Getting Started
The lifecycle has the following simplified steps:
1- createState()
2- mounted == true
3- initState()
4- didChangeDependencies()
5- build()
6- didUpdateWidget()
7- setState()
8- deactivate()
9- dispose()
10- mounted == false
